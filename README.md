# CDR_Analysis

Telecom CDR (Call Detail Record) analysis knowledge base and playbook.

- **[`playbook/`](playbook/README.md)** — the Telecom CDR Analysis Playbook: domain knowledge, field normalization, SIP/dialer-status interpretation, carrier behavior, deliverability best practices, compliance workflows, KPI catalog, and investigation runbooks.
- **[`knowledge_base.json`](knowledge_base.json)** — the structured data layer the playbook references: SIP code table, dialer status table, timezone reference, and the DNC.com call deliverability/compliance guide.

Start at `playbook/README.md` for how the two fit together.
